Laboratorio 1: Módulos en Python

Objetivo del laboratorio

Completar un pequeño programa organizado en dos archivos: mi_modulo.py y main.py. El propósito es practicar módulos estándar, imports, alias, funciones reutilizables y separación del código por responsabilidad.

import from ... import as alias math random datetime módulo propio

Requerimientos de práctica

  1. En mi_modulo.py, importe math, random y datetime.
  2. Cree calcular_area_circulo(radio) usando math.pi.
  3. Cree generar_codigo_estudiante(nombre) usando iniciales y un número aleatorio.
  4. Cree obtener_fecha_actual() usando datetime.
  5. Cree resumen_estudiante(nombre, radio) reutilizando las funciones anteriores.
  6. En main.py, importe el módulo propio como mod.
  7. En main.py, llame a mod.resumen_estudiante() e imprima el resultado.

Recomendación: complete primero mi_modulo.py y después pruebe el resultado desde main.py.
